Simulación de Mercado Libre con JavaScriptEste proyecto es el ejercicio práctico del Laboratorio 03 de la asignatura Desarrollo de Aplicaciones Web de la Escuela Politécnica Nacional. Se trata de una simulación básica de una tienda en línea inspirada en Mercado Libre, donde se muestran productos, se calculan descuentos y se renderiza todo dinámicamente usando JavaScript.
